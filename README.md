# asiris
My attempt to create a messaging app.

Secure, free, and on all platforms.
- Desktop
- Android
- iOS
- Web

## Required Technologies:
- MySQL, preferably installed via `brew`
- Node

## Run and use

### If makefile is available

`make start-db`
and
`make serve-backend` 


then in the other instance/window of terminal:
`make server-frontend`

**Otherwise just copy the commands from the Makefile and paste in the console in the same order**
